Change frame perspective
Change perspective over time to get fake 3D rotation.
Code:
                input='path/to/input.jpg' # image input
output='path/to/output.mp4' # video output

topleft='0:(0+100-(1*in))'
topright='W:(0+100+(1*in))'
bottomleft='0:(H-100+(1*in))'
bottomright='W:(H-100-(1*in))'
duration=50 # in frames

ffmpeg -i $input -vf \
"scale=1280:720,\
loop=$duration:1:1,\
perspective=$topleft:$topright:$bottomleft:$bottomright:sense=source:eval=frame" \
-y $output
Get your console ready...
On line 1,2:
Set input file and output video.

On lines 4-8:
This is where we define each of the 4 corners.

On lines 4:
The first '0' is the x value of the top left corner. Second '0' is the y value of the top left corner. Here we add a 100 to it, and deduct 1 eery frame

On line 37:
Call the filter chain output called [out]. Limit the output duration for 5 seconds. Discard audio. Overwrite output file.
