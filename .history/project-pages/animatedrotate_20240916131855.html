Animated Rotation
Rotate a video from an angle to another angle, over time
Code:
                input=/path/to/input.mp4
output=/path/to/output.mp4
start_angle=0 # angle to start rotating from
angles_to_rotate=70 # how many angles to ratate, strting from start_angle
duration=2 # final output video duration
rotation=($angles_to_rotate/$duration) # calculate how much to rotate per frame

ffmpeg \
-t $duration -i $input \
-vf "rotate='($start_angle*PI/180)+$rotation*(min(t, $duration))*PI/180'" \
-y $output

# Use this filter string if you want to make sure that no edges (background) are exposed. Output video will be cropped and zoomed in.
"rotate='($start_angle*PI/180)+$rotation*(min(t, $duration))*PI/180':ow='min(iw,ih)/sqrt(2)':oh=ow:c=none"
Get your console ready...
The trick here is to calculate how much to rotate, each frame.

To get this number, we first calculate how many angles we need to rotate. endAngle (30) - startAngle (10) = 20.

Then we calculate the duration of the rotation, in frames. $rotateDuration (2) * $frameRate (30) = 60.

Then, we calculate the exact angle to rotate per each frame, so we end at the 'endAngle' value at exactly 'rotateDuration' time.
