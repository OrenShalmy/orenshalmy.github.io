Retro Look
Use this combination of filters to achieve a night vision look.
Code:
                input='path/to/input.mp4'
output='/path/to/output.mp4'

ffmpeg \
-i $input \
-filter_complex \
"[0]scale=1280:720,drawgrid=w=(iw+10)/2:h=(ih+10)/2:t=1:c=white@0.8[cross];
[cross]drawbox=x=10:y=10:w=iw-20:h=ih-20:t=1:color=white@0.9,drawbox=x=20:y=20:w=iw-40:h=ih-40:t=1:color=white@0.9[frame];\
[frame]curves=preset=vintage[look];\
[look]colorize=hue=100[hue];\
[hue]eq=brightness=0.1:gamma=2:contrast=1.4[eq];\
[eq]vignette='PI/4+random(0.3)*PI/100':eval=frame[vignette];\
[vignette]noise=alls=50:allf=t+u[noise];\
[noise]gblur=sigma=2:steps=3[out]" \
Get your console ready...
On line 1,2:
Set input and output video.

On line 4:
Call ffmpeg

On line 5:
Call the input

On line 6:
Start the filter chain

On line 7:
Set final output size and draw a cross shape over the video using the drawgrid filter

On line 8:
Crete a double border around the video using the drawgrid filter, twice

On line 9:
Style the video using the curves filer preset

On line 10:
Shift the hue towards green using the colorize filter

On line 11:
Add some brightness, gamma and contrast using the eq filter

On line 12:
Add vignette using the vignette filter

On line 13:
Add noise using the noise filter

On line 14:
Blur everything using the gblur filter

On line 15:
Map the filter chain to the output, and save the file
