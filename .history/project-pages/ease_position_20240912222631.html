Animated position with easing
Position (overlay) animation of an image over time with easing in/out.
Code:
                framerate=30
canvas=1280x720
loop=30
startX=50
startY=-50
endX=200
endY=50
easing_factor=6

ffmpeg \
-r $framerate -i '/path/to/image.png' \
-f lavfi -i color=color=black:size=$canvas:rate=$framerate:duration=1 \
-y -filter_complex \
"[0]loop='$loop:1:0'[loop];[1][loop]overlay=\
x=$startX+($endX - $startX) * ((exp($easing_factor * (t / ($loop/$framerate))) - 1) / (exp($easing_factor) - 1)):\
y=$startY+($endY - $startY) * ((exp($easing_factor * (t / ($loop/$framerate))) - 1) / (exp($easing_factor) - 1)):eval=frame[o01]" \
-map '[o01]:v' -r $framerate '/path/to/output.mp4'

Set the easing strength and weight. Negative nubers give easing in. Positive give easing out

Call FFMpeg

On line 12:
Create the "canvas" with the dimentions and framerate defines above

On line 14:
