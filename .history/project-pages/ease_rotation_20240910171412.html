Animated Rotation with easing
Rotate a video from an angle to another angle, over time with easing in/out.
Code:
                input='/path/to/input.png'
output='/path/to/output.mp4'
start_angle=0
end_angle=-90
anim_duration=2
easing_factor=-5

ffmpeg \
-loop 1 \
-i $input \
-t $anim_duration \
-vf \
"rotate=angle=(${start_angle} * PI/180) - (((1 - exp(${easing_factor} * (t / ${anim_duration}))) / (1 - exp(${easing_factor}))) * (${start_angle} - ${end_angle}) * PI/180)" \
-y $output
Get your console ready...
On line 1,2:
Define input file and destination file.

On line 3,4:
Define start rotation angle and end rotation angle

On line 5:
Define the total duration of the rotation animation.

On line 6:
Define the ease factor. Negative values will result in easing in. Positive values will result in easing out The higher the value, the faster the easing will be.

On line 8:
Call ffmpeg.

On line 9:
Loop indefinitely over the input file defines earlier. Not needed if the input is video

On line 10:
Call the input file

On line 11:
Set total duration for the animation

On line 12:
Start the filtering

On line 13:
The actual filter. Expressions are used to define the easing parameters

On line 14:
Save to output file
