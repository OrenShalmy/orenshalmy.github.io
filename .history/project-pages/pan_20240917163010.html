Follow an object over time
Follow a region if interest window, from an XY position to an XY position, over time.
Code:
                input='/path/to/input.mp4'
output='/path/to/output.mp4'
fr=24
dur=$((3*$fr))
fromX=100
fromY=20
toX=300
toY=350

ffmpeg \
-ss 01:20 -t 01:23 -i $input -an \
-filter_complex \
"[0]crop=640:360:$fromX+(n*($toX/$dur)):$fromY+(n*($toY/$dur))[out]" \
-map '[out]' -t 3 -an -y $output
