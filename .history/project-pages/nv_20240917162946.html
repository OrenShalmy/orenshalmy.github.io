Retro Look
Use this combination of filters to achieve a night vision look.
Code:
                input='path/to/input.mp4'
output='/path/to/output.mp4'

ffmpeg \
-i $input \
-filter_complex \
"[0]drawgrid=w=(iw+10)/2:h=(ih+10)/2:t=2:c=white@0.5[cross];\
[cross]curves=preset=vintage[look];\
[look]colorize=hue=100[hue];\
[hue]vignette='PI/3.5':mode=forward:aspect=16/9[vignette];\
[eq]noise=alls=50:allf=t+u[noise];\
[vignette]eq=brightness=0.1:gamma=1.7:contrast=1.4[eq];\
[noise]gblur=1[out]" \
-map "[out]" -y $output
Get your console ready...
On line 1,2:
Set input and output video.

On line 4:
Call ffmpeg

On line 5:
Call the input

On line 6:
Start the filter chain

On line 7:
Draw a cross shape over the video with the drawgrid filter

On line 8:
Style the video using the curves filer preset

On line 9:
Shift the hue towards green using the colorize filter

On line 10:
Add vignette using the vignette filter

On line 11:
Add some brightness, gamma and contrast using the eq filter

On line 12:
Add noise using the noise filter

On line 13:
Blur everything using the gblur filter

On line 14:
Map the filter chain to the output, and save the file
